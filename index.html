A Prototype of Microcontroller Based Line Follower AGV Using Arduino

Submitted by: Uday Kumar DL, Vinay Kumar P, Vittal M Gorebal, Yashwanth Gowda

Welcome to our project website! This platform showcases our work on developing an autonomous line-following Automated Guided Vehicle (AGV) using the Arduino microcontroller. Our project aims to address the need for efficient and reliable automated transport solutions in industrial settings, reducing human effort and increasing operational efficiency. Explore the various sections of our detailed project report through the navigation above.

Project Highlights

Autonomous Navigation

The AGV is designed to autonomously follow a black line on a contrasting surface, demonstrating precise path tracking.

Arduino Microcontroller

Utilizes the Arduino Uno as the central processing unit for intelligent control and decision-making.

IR Sensor Array

Employs an array of Infrared (IR) sensors for accurate and real-time line detection and position feedback.

Robust Control

Features refined algorithms and motor control for smooth navigation, even around complex turns and intersections.

Industrial Applications

Addresses needs in manufacturing, warehousing, and logistics for efficient material handling.

Beyond Industry

Potential applications extend to automated valet parking and other smart mobility solutions.
